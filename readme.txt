A board presenting latest tweets.

== Description ==

TweetBoard is a simple widget which lets you present your latest tweets in widget areas.

You can:
<ul>
<li>add twitter username</li>
<li>set whether the date is shown</li>
<li>set the number of shown tweets</li>
<li>edit display colors</li>
</ul>

== Installation ==

1. Extract archive and upload the 'TweetBoard' folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Navigate to 'Plugins' > 'Installed' in the WP admin dashboard. Locate TweetBoard and click activate.
1. Navigate to 'Appearance' > 'Widgets' in the WP admin dashboard. Expand widget to enter options.
